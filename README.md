## REHAU NOVAFLOW Landing

Статический лендинг для `Инсталляции REHAU NOVAFLOW` и панелей смыва.

### Запуск локально
Откройте файл `index.html` в браузере.

### Структура для публикации
- `index.html` — главная страница
- `styles.css` — стили
- `assets/` — изображения

### Развертывание на GitHub Pages
Если будете публиковать из корня репозитория, убедитесь, что в настройках Pages выбран `main` и `root`.
